Introduction¶

Data description:¶

The data contains 110,527 records of patients that showed or not for the appointment. The table contains 14 columns (Characteristics) thay may affect if the patient will show or not.

Columns:¶

  • PatientId : Identification of a patient.
  • AppointmentID : Identification for each appointment.
  • Gender : Female or MAle.
  • ScheduledDay : The day someone called or registered the appointment.
  • AppointmentDay : The day of the actuall appointment.
  • Age : Age of the patient.
  • Neighbourhood : Where the appointment takes place.
  • Scholarship : True of False.
  • Hipertension : True of False.
  • Diabetes : True of False.
  • Alcoholism : True of False.
  • Handcap : True of False.
  • SMS_received : True of False.
  • No-show : (Yes) means if the patient did not show for the appointment, (No) means if the patient show for the appointment.

Data Wrangling¶

Steps:

  • Loading the data.
  • Check for NaN values.
  • Solve the outlier.
  • Change "PatientId" type from ( float ) to ( int ).
  • Change the formate of "ScheduledDay" and "AppointmentDay".
  • Solve Handcap Values.
  • Reset indexes.

Change the formate of "ScheduledDay" and "AppointmentDay"¶

the "ScheduledDay" and "AppointmentDay" formate will be changed from (2016-04-29T18:38:08Z) to (2016-04-29).

  • including only the date of "ScheduledDay" and "AppointmentDay".
  • using to_datetime() function.

** using ( apply ) function instead of ( for loop ) will make the process more faster, and dividing the code into function will also faster the process [ I apply what I have learned during COOP ].

In [18]:
# function to only take the date from a string 
def func(i):
    x= i[:10]
    return x

# using apply function on both "ScheduledDay" and "AppointmentDay" to change their formate 
df["ScheduledDay"] = df.apply(lambda x: func(x["ScheduledDay"]),axis=1)
df["AppointmentDay"] = df.apply(lambda x: func(x["AppointmentDay"]),axis=1)

# change the type from string to dateTime
df["ScheduledDay"] = pd.to_datetime(df["ScheduledDay"])
df["AppointmentDay"] = pd.to_datetime(df["AppointmentDay"])

Conclusion:¶

After answering all the questions above, we can see that the gender does not affect that the patient will show up or not for the appointmet. But we can see that the age can play a role of predicting if the patient will show or not. For example, younger patients have a higher percentage of showing up to their appointments due to that most younger patients may get sick easily as the opposite of older people. Also, a lot of patients do not have scholarship showed up for their appointments. Although sending an SMS message may have the effect of reminding the patient about the appointment, the majority of patients who did not get any SMS messages showed up totheir appointments.
